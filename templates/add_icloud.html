Ajouter un calendrier iCloud

Pour ajouter vos calendriers iCloud, vous devez fournir :

  • Votre identifiant Apple (adresse email)
  • Votre mot de passe spécifique pour les applications

Note importante : Pour des raisons de sécurité, Apple recommande d'utiliser un mot de passe spécifique aux applications plutôt que votre mot de passe principal. Voir comment créer un mot de passe d'application
